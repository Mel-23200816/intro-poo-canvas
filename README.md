# Intro Canvas 2D

Exploración visual e interactiva utilizando la API de Canvas en HTML5, aplicando los principios de la Programación Orientada a Objetos (OOP) con JavaScript. Este proyecto cuenta con un diseño web minimalista, responsivo y de alto contraste.

---

## 🚀 Características del Proyecto

El proyecto se divide en tres tableros interactivos independientes:

* **Objeto 2D Estático:** Demostración de renderizado básico de un objeto instanciado mediante una clase en JS.
* **Objeto con Random:** Implementación de valores aleatorios (coordenadas, colores, tamaños) generados dinámicamente.
* **Múltiples Objetos:** Renderizado simultáneo de varios objetos utilizando arreglos y ciclos, demostrando el potencial de OOP en animaciones complejas.

---

## 🎨 Diseño e Interfaz

* **Paleta de colores:** Negro, Blanco, Gris y un acento en Azul Eléctrico.
* **Layout:** Formato compacto de pantalla única (`100vh`) optimizado para resoluciones de escritorio sin necesidad de scroll vertical.
* **Estructura Modular:** Archivos de estilo (`style.css`) y lógica (`main.js`) separados del documento principal para asegurar la mantenibilidad.
